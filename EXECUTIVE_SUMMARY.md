## 🚀 Como Iniciar

### Setup Rápido

```bash
# 1. Clonar
git clone https://github.com/fomosdeimos-gif/ORAI-Agent.git
cd ORAI-Agent

# 2. Ambiente
python -m venv venv
source venv/bin/activate

# 3. Dependências
pip install -r requirements.txt

# 4. Configurar
cp .env.example .env
# Editar .env com sua chave privada

# 5. Rodar
python main.py

# 6. Dashboard (em outro terminal)
cd frontend
npm install
npm start
```

**URL Dashboard:** http://localhost:3000
